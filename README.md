# FUNCTIONS

## Basic functionalities

* Short answer questions
* Select and multiselect questions
* Flow based on LLM lifecycle
* Output LaTeX code: Can be added as an appendix to a paper
* Output Markdown file: Can be added to their repository
* Register users
* Save progress
* Allow multiple projects by user
* Record usage info of the website
* Show a preview of the output
* Save outputs in the user profile

## Advanced functionalities

* Input a PDF file to extract and prepopulate the form
* User collaboration
* Integration with OSF
* Integration with GitHub: Push updates directly to the user’s repository
* Allow project versioning
